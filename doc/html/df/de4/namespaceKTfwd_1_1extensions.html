Detailed Description

Useful types for implementing fwdpp-based simulations in enviroments like R, Python, etc.

Unlike the rest of fwdpp, functions in this namespace are allowed to throw exceptions, and it is up to the programmer to catch them and handle them appropriately. Both Rcpp and Cython/boost.python

Examples of using this namespace are:

  1. http://github.com/molpopgen/fwdpy (This project is deprecated and no longer tracks changes to the extensions API.)
  2. http://github.com/molpopgen/fwdpy11
